# Khadijah Cakes & Confectionaries

A modern, responsive web application for ordering cakes, pastries, shawarma, pizza, and other delicacies from Khadijah Cakes & Confectionaries.

## Overview

This project is a fully functional e-commerce website that allows customers to browse menu items, search for products, add items to cart, and place orders online.

## Features

✨ **Core Features:**
- Responsive design (mobile-friendly)
- Product catalog with search functionality
- Shopping cart functionality with localStorage persistence
- Real-time price formatting
- Navigation with hamburger menu for mobile devices
- Hero section with call-to-action

🍰 **Products Offered:**
- Cakes (12-inch, 1-layer custom orders)
- Pastries
- Shawarma
- Pizza
- And more confectionaries

## Files Description

### dee.html
- Contains the complete HTML structure
- Navigation bar with logo, search, and cart
- Hero section with value proposition
- Menu section displaying available items
- Order form for customer details
- Uses FontAwesome icons and Google Fonts

### index.js
- Implements search functionality with real-time filtering
- Cart management (add, remove, update quantities)
- localStorage persistence for cart data
- Mobile menu toggling
- Price formatting utilities
- Form handling for orders
- Business contact details (phone and email)

### style.css
- Modern design with chocolate brown color scheme (#261304)
- Responsive grid layouts
- Smooth transitions and animations
- Mobile-first approach with media queries
- Sticky navigation bar
- Hover effects and interactive elements

## Getting Started

1. **Open the project**
   ```
   Open dee.html in your web browser
   ```

2. **No installation required**
   - This is a frontend-only project
   - All dependencies (FontAwesome, Google Fonts) are loaded via CDN
